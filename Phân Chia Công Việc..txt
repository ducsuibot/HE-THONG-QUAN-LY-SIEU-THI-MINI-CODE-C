# NGUYỄN MINH ĐỨC: 
Nhiệm vụ:     

 1. Viết báo cáo tổng quan về hệ thống bằng file word.  

 2. Hàm MENU để người dùng điều khiển chương trình.  

 3. Viết hàm quản lý sản phẩm: 
	+ Them san pham. 
	+ Sua san pham. 
	+ Xoa san pham. 
	+ Tim san pham theo ma. 
	+ Tim san phan theo ten.  
	+ Sap xep san pham theo ma.
	+ Sap xep san pham theo ten. 
	+ Luu san pham vao file.   
	+ Hien san pham tu file.
